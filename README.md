# NoteFlow — SaaS Notes Dashboard

A modern and clean notes management dashboard built with React.

## 🚀 Features

* Create, edit and delete notes (CRUD)
* Real-time search functionality
* Local storage persistence
* Responsive design
* Dashboard statistics
* Clean and modern UI

## 🛠️ Tech Stack

* React
* JavaScript
* CSS
* LocalStorage
